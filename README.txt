The R script 'phyloDPM.R' implements the model described in the paper "Associating the microbiome with host phenotype using Dirichlet process random effects model". 
The three datasets used in the simulation are available on the links:

Schizophrenia data:
(https://qiita.ucsd.edu) with study ID 11710

HIV/AIDS data:
(https://qiita.ucsd.edu) with study ID 11669.

Atherosclerosis data: 
(https://qiita.ucsd.edu)  with study ID 349,


The simulations are contained  in the R script 'simulation.R'
